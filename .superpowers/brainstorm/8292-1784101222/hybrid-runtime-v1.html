路径 A · 普通知识点
AUTHOR TRACE

经过验证的预设运行轨迹

  1. 读取课程内的结构化 frames
  2. 按时间线播放队列、模块、流等状态
  3. 同步输出终端日志和解释

优势：无网络依赖、速度快、结果稳定，教师可以精准强调关键步骤。

ISOLATED SANDBOX

隔离环境真实执行

  1. 服务端按 lessonId 加载可信项目模板
  2. 应用学习者选择的受控代码补丁
  3. 放入独立沙箱执行并收集事件

边界：浏览器不上传任意 Node.js 源码；Next.js 进程不直接执行课程代码。

统一事件格式包含：时间、阶段、事件类型、可视化通道、状态数据、日志和教学注释。前端不需要知道事件来自预设轨迹还是真实沙箱。

沙箱安全约束

  • 默认禁止访问外部网络，仅开放必要的内部回环
  • 临时工作目录、只读运行时、任务结束立即清理
  • 限制 CPU、内存、运行时间、进程数和输出大小
  • 依赖包预安装并使用白名单，不在运行时执行 npm install
  • 所有密钥与宿主环境变量对沙箱不可见

错误与降级策略

  • 错误答案不触发沙箱，只进入定向教学反馈
  • 语法或运行错误作为可学习事件展示堆栈位置
  • 超时、内存超限、输出超限分别显示明确原因
  • 沙箱服务不可用时回退到标记为“模拟”的验证轨迹
  • 切换课程或取消运行时终止任务并丢弃迟到事件
实现边界

首期：只执行官方课程提供的项目模板和有限补丁。以后：若增加自由代码编辑器，必须作为独立安全项目重新设计，不复用当前受控补丁接口。
