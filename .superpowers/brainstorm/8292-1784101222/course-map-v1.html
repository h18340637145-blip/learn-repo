设计第 1 节：10 阶段能力递进课程地图

主线约 80 个知识点，每阶段 7–9 课 + 1 个项目。官方主题被重新按依赖顺序编排，高级内容进入选修轨道。

01 · 运行时与命令行

认识 Node.js

运行时、浏览器差异、V8、版本管理、脚本、REPL、process、环境变量

项目：CLI 系统信息探测器

02 · 模块与包

组织可维护代码

ESM、解析规则、package.json、npm、语义化版本、CommonJS、缓存、TypeScript 入门

项目：依赖与配置检查器

03 · 异步与事件

建立事件循环心智模型

回调、Promise、async/await、计时器、微任务、nextTick、EventEmitter、取消任务

项目：并发任务调度器

04 · 文件、Buffer 与 Stream

高效处理本地数据

path、URL、fs/promises、目录与元数据、Buffer、编码、流、pipe、背压、readline

项目：CLI 日志分析器

05 · HTTP 基础

理解一次网络事务

Server、请求响应、Header、状态码、路由、查询参数、请求体、流式响应、Fetch

项目：流式静态文件服务器

06 · API 与服务设计

构建可靠后端接口

REST、输入验证、错误模型、配置边界、日志、超时、AbortSignal、优雅关闭

项目：任务管理 REST API

07 · 进程与并发

突破单主线程边界

阻塞识别、libuv 线程池、child_process、worker_threads、消息传递、Cluster、并发模型选择

项目：Worker Pool 报表生成器

08 · 实时通信

把服务变成实时系统

长轮询、SSE、WebSocket、心跳、连接状态、广播、背压、断线恢复

项目：实时任务通知服务

09 · 测试与安全

用证据保证正确性

node:test、断言、Mock、覆盖率、集成测试、权限模型、依赖安全、常见攻击防护

项目：经过测试的鉴权服务

10 · 诊断与生产工程

让服务可观测、可上线

Inspector、性能分析、内存快照、GC、火焰图、健康检查、指标、生产配置、发布检查

项目：生产故障诊断实验室

最终综合项目 · 实时协作任务平台

模块边界 → REST API → 实时同步 → Worker 任务 → 鉴权安全 → 自动化测试 → 性能诊断 → 生产运行

选修轨道

Node-API 原生扩展、WebAssembly、HTTP/2 与 TLS 深入、Linux perf、高级内存诊断。选修不阻塞主线毕业。
